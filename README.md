# Gant-tasks

Интерактивный редактор плана работ с диаграммой Ганта (drag & drop). Один HTML-файл, без сборки и сервера.

## Быстрый старт

1. Скачайте или клонируйте репозиторий.
2. Откройте `gantt-interactive-editor.html` в **Chrome** или **Edge**.
3. Настройте дату старта, оценки по блокам и перетаскивайте задачи на диаграмме.

## Возможности

- Диаграмма Ганта с перетаскиванием и изменением длительности задач
- Таблица оценок по функциональным блокам (MD) с синхронизацией с диаграммой
- Глобальные настройки: дата старта, точка сдвига параллельных задач, масштаб (дни / недели / месяцы)
- Импорт и экспорт плана в JSON
- Прямая работа с файлом через File System Access API (автосохранение)
- Локальное автосохранение в `localStorage`
- История изменений и отмена последнего действия
- Сравнение с базовым планом (418 MD)

## Требования

- Современный браузер (Chrome / Edge рекомендуются для работы с файлами на диске)
- Подключение к интернету для загрузки [Frappe Gantt](https://github.com/frappe/gantt) с CDN
